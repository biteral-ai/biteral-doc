# Brand

Represents one of your products' brands in Biteral. In addition to the data you provided about the brand, it includes Biteral's own information, such as the brand's internal identifier or its processing status.

Property|Type|Description
---|---|---
<Badge variant="sdk php property" icon="property" text="id" />|string|Brand ID in Biteral. For example: <Badge variant="value" text="bra_hs14jd193K1ndJ" />
<Badge variant="sdk php property" icon="property" text="createdAt" />|DateTimeImmutable|Brand creation date.
<Badge variant="sdk php property" icon="property" text="updatedAt" />|DateTimeImmutable|Date when the brand was last updated.
<Badge variant="sdk php property" icon="property" text="payload" />|<Badge variant="sdk php payload" text="BrandPayload" to="/php-sdk/payloads/brand-payload" />|Payload of the brand data
