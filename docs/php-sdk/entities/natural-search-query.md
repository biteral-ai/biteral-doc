# NaturalSearchQuery

Representa una petición de búsqueda para el [buscador natural](https://biteral.ai/natural-search).

### Propiedades

||
---|---|---
<Badge icon="screen-full" text="query" />|string|La cadena escrita por el usuario para encontrar productos en lenguaje natural, por ejemplo `Un regalo para alguien a quien le encanta cocinar`
<Badge icon="screen-full" text="limit" />|int|El número máximo de productos a obtener, por ejemplo 5. Por defecto, se obtienen 10 productos como máximo.
