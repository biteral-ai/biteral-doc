# NaturalSearchQuery

Represents a search query for the [natural search engine](https://biteral.ai/natural-search).

Property|Type|Description
---|---|---
<Badge variant="sdk php property" icon="property" text="query" />|string|The string typed by the user to find products in natural language, for example <Badge variant="value" text="A gift for someone who loves cooking" />
<Badge variant="sdk php property" icon="property" text="limit" />|int|The maximum number of products to return, for example 5. By default, a maximum of 10 products are returned.
