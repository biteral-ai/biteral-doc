# Category

Represents one of your products' categories in Biteral. In addition to the data you provided about the category, it includes Biteral's own information, such as the category's internal identifier or its processing status.

Property|Type|Description
---|---|---
<Badge variant="sdk php property" icon="property" text="id" />|string|Category ID in Biteral. For example: <Badge variant="value" text="cat_hs14jd193K1ndJ" />
<Badge variant="sdk php property" icon="property" text="createdAt" />|DateTimeImmutable|Category creation date. For example: <Badge variant="value" text="prj_18vn8R74uvSijf" />
<Badge variant="sdk php property" icon="property" text="updatedAt" />|DateTimeImmutable|Date when the category was last updated.
<Badge variant="sdk php property" icon="property" text="payload" />|<Badge variant="sdk php payload" text="CategoryPayload" to="/php-sdk/payloads/category-payload" />|Payload of the category data
