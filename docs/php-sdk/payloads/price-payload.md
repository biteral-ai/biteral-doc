# PricePayload

Represents a price of one of your products.

Property|Type|Description
---|---|---
<Badge variant="sdk php property" icon="property" text="amount" />|string|The price, with an arbitrary number of decimal places, using the `.` symbol as the decimal separator. For example: <Badge variant="value" text="24.95" />
<Badge variant="sdk php property" icon="property" text="currency" />|string|The currency in which the price is expressed, according to the [ISO 4217](https://en.wikipedia.org/wiki/ISO_4217) standard. For example <Badge variant="value" text="EUR" />
